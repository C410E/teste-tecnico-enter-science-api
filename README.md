# API da Aplicação de Contratação de Artistas

Para rodar a API, entre na pasta da API com `cd api`, instale as dependências executando `composer install`, em seguida inicie os containers com `docker compose up -d`. Depois, rode as migrations com `php artisan migrate` e inicie o servidor com `php artisan serve`. A API estará disponível no endereço indicado pelo `php artisan serve`, normalmente `http://127.0.0.1:8000`.
